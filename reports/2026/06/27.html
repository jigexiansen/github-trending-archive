TypeScript edges out Python today, carried almost entirely by design.md's runaway lead — a project that, fittingly, is itself a spec for teaching coding agents taste. Five of the ten entries explicitly target AI agent workflows, from document parsing to investment research to a Chinese career-advisor skill, while old-guard infra names like Grafana and openpilot post modest single-digit gains. The pattern holds: tooling for agents is currently outpacing tooling agents merely run on.

TypeScript今天小胜Python，几乎全靠design.md的遥遥领先撑起——颇为巧妙的是，这个项目本身就是一份教编程智能体「懂审美」的规范文件。十个项目里有五个明确瞄准AI智能体工作流，从文档解析到投资研究，再到一个张雪峰风格的职业规划Skill；而Grafana、openpilot这类老牌基建项目只录得个位数的温和增长。趋势很清晰：服务于智能体的工具，正在跑赢智能体单纯运行其上的工具。
