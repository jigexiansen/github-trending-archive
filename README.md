# 📈 GitHub Trending Archive

Daily snapshots of [GitHub Trending](https://github.com/trending), auto-generated as standalone HTML reports with visual analytics.

## 📁 Structure

```
{YYYY}/
└── {MM}/
    └── github-trending-{YYYY-MM-DD}.html
```

Each file is a self-contained HTML page — no build step, no dependencies. Open directly in any browser.

## ✨ Report Features

- Top 10 trending repos of the day
- Today's star counts, total stars, language breakdown chart
- Trend insight summary (in Chinese)
- Unique visual style each day (editorial / minimal / dashboard / brutalist / etc.)

## 🔖 Quick Access

Browse by year → month in the directory tree above, or clone and open locally:

```bash
git clone https://github.com/jigexiansen/github-trending-archive
open 2026/05/github-trending-2026-05-28.html
```
